\begin{Exec}
  证明空间$\bR\times\bR^3$的每个加利略变换都可唯一得表示为一个旋转, 
  一个平移, 一个匀速运动的组合
  ($g = g_1 \circ g_2 \circ g_3$ )(所以, 加利略变换群的维数是$3+4+3=10$)
\end{Exec}

\begin{proof}
  对任意加利略变换$g$, 先考虑$g(t,\b0)$由$g$是线性的, 可知, 有$\bv$, $\ba$,$s$, 
  使得$g(t,\b0) = (t-s, \ba + t\bv) \in \bR\times\bR^3$.
  下面考虑$\bR^3$中的标准正交基$\be_1,\be_2,\be_3$.
  令$g(t,\be_i) - g(t,\b0)=  (t-s,\bu_i)$.
  由于同时事件空间的差是一个3维线性空间(定义).
  
  所以$g(t,(a,b,c)) - g(t,\b0)= (t-s, G(a,b,c)^T)$,
  其中$G = (\bu_1,\bu_2,\bu_3)$.
  令$g_3(t,\bx)  = (t,G \bx)$,
  $g_2(t,\bx) = (t,t \bv +\bx)$, $g_1(t,\bx) = (t-s,\ba+\bx)$.
  则$g = g_1\circ g_2\circ g_3$, 即为所求.
  
\end{proof}

\begin{Exec}
  证明一切加利略空间均互相同构, 而特别是同构于坐标空间$\bR\times\bR^3$.
\end{Exec}

\begin{proof}
  由于$t:\bR^4\rightarrow \bR$, 是一个 Hilbert 空间上的线性范函,
  有如下分解:$\bR^4 = \ker{t}\oplus\ker{t}^\perp$, 且$\ker{t}$的维数是3.
  设$P,Q$为对应的投影映射, 使$\bx = P\bx+Q\bx$, 其中$P\bx \in\ker{t}$,
  $Q\bx \in\ker{t}^\perp$. 又令$R$为$\ker{t}$到$\bR^3$的一个等距同构. 
  
  任取$A^4$中一点$\bx_0$.
  可作$F: A^4 \rightarrow \bR\times\bR^3$, 其中$F(\bx) = (t(\bx-\bx_0),R\,P\bx)$.
  即为所求同构.
\end{proof}

\begin{Exec}
  若一力学系只由一点组成, 证明它在惯性参考系中的加速度为零.
\end{Exec}
\begin{proof}
  由例1,例2,加速度不依赖于$\bx,\bxd$和$t$, 所以$F(\bx,\bxd,t)=\bv$,
  又由旋转不变性对任意$G$, $\bv=F(G\bx,G\bxd,t)=GF(\bx,\bxd,t)=G\bv$
  所以只能有$\bv=0$. 即加速度为零.
\end{proof}

\begin{Exec}
  设一个力学系由两点组成, 在初始时刻其速度均为零, 
  证明他们将停留在初始时刻联结他们的直线上.
\end{Exec}

\begin{proof}
  只要证明$\bxdd_i$平行于$\bx_1-\bx_2$. 
  显然$\bx_1-\bx_2$非零, 可令$\bu=(\bx_1-\bx_2)/\left\|\bx_1-\bx_2\right\|$, 
  对任意与$\bu$垂直的单位向量$\bv$,令$\bw = \bu\times\bv$ 
  令$G = (\bu, \bv,\bw)^T$, 为一个旋转变换.
  由例3的相同方法,可知$F=\bxdd = (\bxdd_1,\bxdd_2)$
  只依赖于$\bx_1-\bx_2$和$\bxd_1-\bxd_2$.
  则$G\bxdd = G\,F(\bx_1-\bx_2,\bxd_1-\bxd_2,t) 
  = F(G\bu,G\b0,t) \equiv F((\|\bu\|,0,0)^T,\b0,t)$.
  由$\bv$的任意性, $\bxdd_i$垂直于$\bv$. 又考虑到$\bx_1 = \bx_2 = 0$
  所以$\bx_i(t)$始终在他们的连线上.
  \end{proof}

\begin{Exec}
  设一个力学系由三点组成, 在初始时刻其速度均为零,
  证明它们总位于初始时刻包含它们的平面上.
\end{Exec}

\begin{proof}
  类似前一题, 构造正交变换$G$.
  令$\bu$是垂直于$\bx_1-\bx_2,\bx_1-\bx_3$的向量.
  $\bv$平行于$\bx_1-\bx_2$,
  $\bw=\bu\times\bv$,
  $G = (\bu,\bv,\bw)$,$G' = (-\bu,\bv,\bw)$,$\bx = (\bxdd_1,\bxdd_2,\bxdd_3)$.
  由书中例, $F$只与$\bx_1-\bx_2,\bx_1-\bx_3$有关. 
  则($\star$中部分不必考虑具体形式)
  $$G\bxdd=F(G(\bx_1-\bx_2,\bx_1-\bx_3),G(\bxd_1,\bxd_2,\bxd_3),t)
  = F((0,0,0;\star),\b0,t) = G' \bxdd$$,
  得$\bu\bxdd = \b0$, 同上题得证.
\end{proof}

\begin{Exec}
  设一个力学系由两点组成, 证明, 对任意初始条件都存在一个惯性参考系, 使得其中这两个点总位于一个固定平面上.
\end{Exec}

\begin{proof}
  设两点初始时刻位置和速度分别为$\bx_1,\bx_2$和$\bxd_1,\bxd_2$, 
  考虑一个以$\bxd_1$运动的参考系, 在其中$\bxd_1=\b0$.
  考虑单位向量$\bu$同时垂直于$\bx_1-\bx_2$和$\bxd_2$. 
  另作单位向量$\bv,\bw$, 使它们成为两两正向量. 令$G = (\bu, \bv,\bw)$, 
  $G' = (-\bu,\bv,\bw)$. $\bxdd = (\bxdd_1,\bxdd_2)$.
  同时, $F$ 只与$\bx_1-\bx_2$, $\bxd_1$,$\bxd_2$有关.
  于是
  $$G\bxdd = F(G(\bx_1-\bx_2),G\bxd_1,G\bxd_2)
  = F((0,\star)^T, \b0,(0,\star))
  = F(G'(\bx_1-\bx_2),G'\bxd_1,G'\bxd_2) = G'\bxdd
  $$
  于是$\bxdd$和$\bu$垂直, 结合$\bxd_1,\bxd_2$和$\bu$垂直即得,
  运动始终在和$\bu$垂直的平面内.
\end{proof}

\begin{Exec}

\begin{proof}

    \begin{array}{ccc}

    \end{array}
  \right)
  $$
  即得.
\end{proof}

\begin{Exec}
  惯性参考系类是否唯一
\end{Exec}

\begin{proof}
  将惯性参考系类理解为, 若两个参考系为同一类, 则它们之间有一个加利略变换. 
  那么改变时间间隔或者时间方向的两个惯性参考系之间是没有加利略变换的.
  若有, 则这个变换不保持时间间隔数$t$, 所以不是加利略变换.
\end{proof}
